    # NVM介绍

    NVM全称node.js version management ，专门针对node版本进行管理的工具，通过它可以安装和切换不同版本的node.js

    # 使用场景

    一些老项目用的是vue2.5左右,webpack版本也比较低，只能使用10.16.0左右的node版本，但是也有一些新项目需要使用高版本的node例如14.17.3左右的这时候就可以使用nvm切换node 版本

    # 1.安装nvm

    # windows github (opens new window)（程序员交友平台）地址

    Releases · coreybutler/nvm-windows · GitHub (opens new window)

    # Mac

    1.执行以下脚本

    sudo curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.38.0/install.sh | bash
    1

    2.重启电脑 或者是刷新bash文件即可生效

    source ~/.bash_profile
    1

    # Nvm 常用命令操作

    查看现在所有安装的node版本

    nvm list
    1

    查看nodejs 官方的所有版本

    nvm list available
    1

    下载对应的node版本号

    nvm install （版本号)
    1

    切换node版本

    nvm use
    1

    # NRM介绍

    nrm 是一个 npm 源管理器，允许你快速地在 npm源间切换。

    1.安装

    npm install -g nrm
    1

    2.查看可选源 星号代表当前使用源

    nrm ls
    1

    若出现一下错误

    nrm ls const open = require('open');

    这是因为nrm依赖于一个名为open的包。

    使用此命令可以修复此错误

    npm install -g nrm open@8.4.2 --save
    1

    3.切换源

    nrm use （源）切换源
    1

    4.添加源

    nrm add （名称）（源）   格式和前面列出来的源的格式一致
    1

    5.测速

    nrm test npm
    1
